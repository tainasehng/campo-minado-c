# Campo Minado em C

Implementação do jogo Campo Minado em linguagem C, com funcionalidades completas de jogo em terminal.

## Funcionalidades
- Geração aleatória de bombas
- Sistema de níveis de dificuldade
- Revelação de células com expansão automática
- Controle de tempo de jogo
- Sistema de pausa e retomada

## Tecnologias
- Linguagem C

## Como executar

1. Compile o código:
```bash
gcc campo.c -o campo
```

2. Execute:
```bash
./campo
```

## Objetivo

Praticar lógica de programação, manipulação de matrizes e organização de código em linguagem C.
